Networking: CREATING VIRTUAL LANS

A networking project focused on virtual LAN setup, segmentation, and configuration.

Description

This project demonstrates the creation and configuration of Virtual Local Area Networks (VLANs) using multilayer switches. Hosts were assigned to separate VLANs to segment network traffic and improve organization.

An 802.1Q trunk link was configured between switches to allow communication between devices in the same VLAN across multiple switches while maintaining isolation between different VLANs.

The project demonstrates VLAN segmentation, trunking, switch configuration, and network management concepts
