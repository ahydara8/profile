Object Detection Using Arduino

Smart object detection system using an ultrasonic sensor, LCD display, and LED status indicators.

Description

This project is a smart object detection system using an Arduino and ultrasonic sensor to measure distance and display real-time values on an LCD screen while providing visual feedback through LEDs.

  • When distance measurement range is 2–8 inches, Red LED is ON.
  • When distance measurement range is 9–15 inches, Yellow LED is ON.
  • When distance measurement range is 16–22 inches, Green LED is ON.

A push button is used to enable or disable the system, and a potentiometer is used to adjust LED brightness or system sensitivity. This system has real-life application in vehicle parking assistance to help prevent collisions.
